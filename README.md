# RoadCafe - Симулятор управления с элементами градостроительства и экологии (есть мини-игра с белкой)
Студент: Непсов Андрей Львович

Core (Системы)
DayManager - Управление игровым циклом: длительность дня, переход между днями, сохранение прогресса через синглтон, отображение отчёта в конце дня;
Economy -	Экономическая система: стартовый капитал, доходы от продажи еды, расходы на улучшения, события изменения баланса;
Ecology -	Система загрязнения: пассивное уменьшение со временем, влияние мусора и стационарных объектов, события изменения уровня.

Characters (Персонажи)
PlayerPickup -	Управление предметами в руках игрока: подбор (ЛКМ), бросание (ПКМ), передача предметов в производство;
CustomerBrain -	Поведение покупателей: маршрут (точка еды → касса → столик → выход), ожидание еды и обслуживания, генерация мусора;
CustomerMovement -	Навигация покупателей через NavMeshAgent, остановка по достижении цели.

Interactable (Интерактивные объекты)
Production -	Производство предметов: кухня (требует ингредиенты), грядка (автоматическое производство);
Storage -	Хранение предметов с визуальным отображением иконок;
CashRegister -	Обслуживание покупателей: появление иконки при подходе игрока, начисление денег;
TrashBin	- Хранение мусора и вызов мусоровоза;
Filter	- Постройка фильтра очистки для постоянного снижения загрязнения.

Items (Предметы)
Item -	Базовый класс всех предметов: подбор, бросание, состояние (в руках/на земле);
CookedFood -	Готовая еда. Результат работы кухни;
CrateVegetables -	Ящик с овощами. Производится грядкой;
TrashBag -	Мешок с мусором, разлагается за 30 секунд, увеличивает загрязнение.

UI (Интерфейс)
GameUI -	Отображение текущих денег и уровня загрязнения (слайдер, цветовая индикация, мигание при критическом уровне);
DayReport -	Панель отчёта в конце дня: доходы, расходы, изменение загрязнения, статус поимки белки;
DayTimerUI -	Таймер обратного отсчёта дня, цветовое изменение при окончании времени.

Environment (Окружение)
Squirrel -	Белка: бегает вокруг дерева, реагирует на клики (после 30 кликов попадает в руки игрока);
Tree -	Визуальное изменение дерева (листва исчезает при загрязнении >50%);
EcologyAffect -	Постоянное влияние объектов на экологию.

Helpers (Вспомогательные)
GameStarter -	Создание Canvas каждый игровой день;
IconClickHandler -	Обработка кликов по иконкам действий.

Архитектурные решения
Синглтоны: DayManager, Economy, Ecology, GameUI
Сохранение состояния: DontDestroyOnLoad для критических систем
События: обновление UI через подписку на события
Модульность: скрипты разделены по ответственности (движение, логика, UI)

Технологии
Unity 2022.3+
NavMesh
Cinemachine
PlayerPrefs
